# Notes
A basic Command line To do notes app created using Javascript.

### To add a note
`
node app.js add -t "write title here" -b "write body here"
`
### To remove a note
`
node app.js remove -t "write title here" 
`
### To read a note
`
node app.js read -t "write title here"
`
### To list all notes 
`
node app.js list
`
