# Notes
A Command line notes app created using javascript

### To add a 
`
node app.js add -t "write title here" -b "Write body here"
`
### To remove a note
`
node app.js remove -t "write title here" 
`
### To read a note
`
node app.js read -t "write title here"
`
### To list all notes 
`
node app.js list
`
